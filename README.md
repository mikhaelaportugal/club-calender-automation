# Google Form → Google Calendar Automation

This Google Apps Script automatically adds events to a Google Calendar
when a member submits a Google Form.

## Features
- Creates calendar events based on form responses
- Automatically sets date, time, title, and description
- Uses onFormSubmit trigger

## Files
- Code.gs — main automation logic
- TriggerSetup.gs — installs onFormSubmit trigger
- appsscript.json — Apps Script project settings
